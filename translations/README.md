## Contribuer

Pour ajouter une traduction :
1. Creer un fichier dans ce dossier avec le meme numero de prefixe que le guide source.
2. Adapter le contenu au contexte francophone ouest-africain (administrations, secteur public, infrastructures critiques).
3. Ouvrir une Pull Request en referencant le guide source.
